## Project Overview
This project analyzes customer purchasing behavior and product performance using SQL. 
The dataset includes orders, products, aisles, and departments to derive actionable business insights.

## Dataset Used
- Orders
- Products
- Aisles
- Departments

## Key Business Questions Answered
- Which departments generate the highest number of orders?
- Top-performing products based on order frequency
- Customer reordering behavior analysis
- Aisle-level performance insights

## SQL Concepts Used
- JOINs (INNER, LEFT)
- GROUP BY & HAVING
- Subqueries
- Window Functions
- Aggregate Functions
- CASE statements

## Insights & Outcomes
- Identified high-demand departments to support inventory planning
- Analyzed repeat purchase behavior for customer retention insights
- Highlighted top products contributing to overall sales volume

## Tools
- SQL (MySQL / PostgreSQL)
- GitHub

## How to Use
1. Download the `.sql` file
2. Load dataset into SQL database
3. Run queries sequentially
